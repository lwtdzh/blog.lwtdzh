# Agent Instructions

This repository keeps shared reusable agent workflows in `.skills/`.

When a task looks like a reusable project workflow, inspect `.skills/**/SKILL.md` and use the most relevant skill.

When regression testing this blog, load and follow `.skills/blog-regression-checks/SKILL.md`.

That file is the canonical regression checklist for this repository.

Do not duplicate, fork, or silently drift from `.skills/blog-regression-checks/SKILL.md` in tool-specific instruction files unless the repository owner explicitly asks for it.
